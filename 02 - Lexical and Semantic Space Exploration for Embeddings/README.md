#  Lexical and Semantic Space Exploration for Embeddings

## Introduction

This project explores the generation and analysis of sparse and dense word embeddings for the **Marathi language**. Using a large-scale Wikipedia corpus, this study implements and compares two prominent methods based on word co-occurrence statistics:

  - **COALS (Correlated Occurrence Analogue to Lexical Semantics)** for generating sparse word vectors.
  - **Singular Value Decomposition (SVD)** on a Pearson correlation matrix to create dense word vectors.

The core objective is to analyze the semantic quality of the resulting embeddings and investigate the impact of retaining negative correlations on vector representations.

### Dataset

The corpus for this project was derived from a Marathi Wikipedia dump, containing over **15 million tokens**. Due to its large size, the processed corpus is not included in this repository.

## Problem Statement

The project is designed to generate and evaluate word vectors from a large linguistic corpus. The primary tasks are:

1.  [cite\_start]**Co-occurrence Matrix Construction:** Build a word co-occurrence matrix ($M = |V| \times |V|$) from the Marathi corpus, limiting the vocabulary based on machine capability. [cite: 53, 54, 55]
2.  [cite\_start]**Pearson Correlation Computation:** Implement a parallelized process to compute the Pearson correlation matrix from the co-occurrence data. [cite: 70, 84, 85]
3.  [cite\_start]**Sparse Embedding (COALS):** Generate sparse word vectors using the COALS algorithm, which involves normalizing the positive entries of the correlation matrix. [cite: 86]
4.  [cite\_start]**Dense Embedding (SVD):** Apply Singular Value Decomposition (SVD) to the Pearson correlation matrix (including negative values) to create dense, lower-dimensional word vectors. [cite: 98, 99]
5.  [cite\_start]**Similarity Analysis:** For both sparse and dense embeddings, identify the top 5 most similar words for a target set of 5 nouns, 5 verbs, and 5 adjectives using cosine distance. [cite: 88, 91, 103, 104]
6.  [cite\_start]**Comparative Analysis:** Analyze and report on the effectiveness of the two approaches, specifically focusing on how retaining negative correlations in the SVD method impacts the quality of the embeddings. [cite: 109, 110]

## Solution and Approach

The solution is implemented in a Jupyter Notebook (`AyushmanMDS202411_Assignment02.ipynb`) and follows these steps:

1.  **Data Preprocessing and Tokenization:** The Marathi corpus was tokenized using the `indicnlp` library. The vocabulary was limited to the **top 5,000 most frequent tokens** to create a $5000 \times 5000$ co-occurrence matrix.
2.  **Parallelized Correlation Matrix:** The Pearson correlation matrix was computed using a function that was parallelized across all available CPU cores with Python's `joblib` library to improve performance.
3.  **Sparse Embeddings (COALS):** The COALS word vectors were derived by taking the Pearson correlation matrix, setting all negative values to zero, and normalizing the rows.
4.  **Dense Embeddings (SVD):** SVD was applied directly to the Pearson correlation matrix. Analysis showed that **1108 dimensions** were sufficient to capture **99.99% of the variance**, providing an effective dimensionality reduction.
5.  **Similarity Measurement:** For both sets of embeddings (sparse and dense), cosine distance was used to find the top 5 similar words for 15 target words (nouns, verbs, and adjectives identified using `stanza`). Results were tabulated with English translations provided by `googletrans`.
6.  **Analysis and Findings:** A comparative analysis was performed by examining the overlap in similar words generated by both methods and reflecting on the semantic coherence of the results.

## Results

  - **Similarity Tables:** For each of the 15 target words, tables were generated showing the top 5 similar words and their cosine distance scores from both the sparse (COALS) and dense (SVD) embeddings.
  - **Superiority of Dense Embeddings:** The dense embeddings generated via SVD (which retained negative correlations) consistently produced more semantically coherent and contextually relevant similar words. For example, for the noun **`क्रिकेट` (Cricket)**, the dense vectors found related terms like **`खेळणारा` (Playing)** and **`खेळलेला` (Played)**, which were missed by the sparse model.
  - **Impact of Negative Correlations:** The analysis concluded that retaining negative correlations is crucial for capturing richer semantic relationships. These anti-associations help the model better differentiate between words, leading to higher-quality embeddings.
  - **Dimensionality Reduction:** The SVD variance plot clearly illustrates that a dense vector size of **1108** is an optimal choice for retaining nearly all information from the original $5000 \times 5000$ matrix.
